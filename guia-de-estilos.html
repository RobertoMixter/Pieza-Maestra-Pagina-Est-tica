Botones

Botón Primario Usado para las llamadas a la acción más importantes (Ej: Contactar).
Botón Secundario Usado para acciones secundarias (Ej: Ver todos los productos).
Botón Terciario Usado para acciones menos prominentes, con estilo de contorno.

Efecto Hover: Todos los botones cambian a un fondo amarillo eléctrico para máxima visibilidad.
